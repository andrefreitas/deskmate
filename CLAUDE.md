# Deskmate — Coworking Space AI Agent

## Project Overview

Deskmate is an AI agent for managing a coworking space. It handles desk allocation, bookings, and related operational tasks through a conversational interface.

## Core Features (planned)

- **Desk allocation** — assign, release, and reassign desks to members
- **Booking management** — reserve desks by day, half-day, or custom time slots
- **Member management** — track active members, preferences, and history
- **Space overview** — real-time view of desk availability and occupancy
- **Notifications** — remind members of upcoming bookings or expirations

## Agent Design

The agent should be able to:
- Understand natural language requests (e.g. "book me a quiet desk for tomorrow afternoon")
- Query and mutate the desk/booking state
- Handle edge cases: no availability, conflicting bookings, cancellations
- Respond clearly and concisely
